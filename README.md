# Number of zeroes in 8 bit input

This verilog code counts the number of zeroes in an 8 bit input.  
Example - 11111100 has 2 zeroes, the output will be displayed as 2.  

Output sample:  
11011101: 2  
11011110: 2  
11011111: 1  
11100000: 5  
11100001: 4  
11100010: 4  
11100011: 3  
11111111: 0
